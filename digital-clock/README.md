# Digital Clock - Multiple Time Zones

A modern, interactive digital clock that displays the current time across 8 major world time zones plus your local time. Perfect for students, professionals, and global teams!

## Features

⏰ **Real-Time Updates**
- Live clock displays update every second
- Shows time in 9 different time zones simultaneously
- Automatic daylight saving time adjustment

🌍 **World Time Zones**
- New York (EST/EDT)
- London (GMT/BST)
- Paris (CET/CEST)
- Dubai (GST)
- Tokyo (JST)
- Sydney (AEDT/AEST)
- Los Angeles (PST/PDT)
- Singapore (SGT)
- Your Local Time Zone

🎛️ **Customizable Display**
- Toggle between 12-hour and 24-hour format
- Show/hide seconds display
- Keyboard shortcuts (T for time format, S for seconds)

🎨 **Modern Design**
- Beautiful gradient background
- Glass-morphism effect on clock cards
- Smooth animations and transitions
- Fully responsive design (mobile, tablet, desktop)
- Cyberpunk-inspired neon aesthetics

📅 **Additional Information**
- Shows current date with day name
- Displays timezone code and UTC offset
- Highlights local time with different styling

## How to Use

1. Open `index.html` in your web browser
2. View current times across all time zones
3. Click "Toggle 12/24 Hour" to switch time format
4. Click "Toggle Seconds" to show/hide seconds
5. Use keyboard shortcuts:
   - **T** - Toggle 12/24 hour format
   - **S** - Toggle seconds display
