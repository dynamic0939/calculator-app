# School Calculator

A simple, user-friendly calculator application designed for school students. This calculator provides basic arithmetic operations with a clean, intuitive interface.

## Features

✨ **Easy to Use Interface**
- Large, readable display
- Clear button layout
- Color-coded buttons for different operations

🧮 **Basic Operations**
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Decimal support (.)

⌨️ **Keyboard Support**
- Number keys (0-9)
- Operator keys (+, -, *, /)
- Enter to calculate
- Backspace to delete last digit
- Escape to clear all

🎨 **Modern Design**
- Beautiful gradient background
- Responsive design for all devices
- Smooth animations and transitions
- Error handling (division by zero)

## How to Use

1. Open `index.html` in your web browser
2. Click buttons or use your keyboard to enter numbers and operations
3. Press `=` or `Enter` to get the result
4. Use `C` button or `Escape` key to clear
5. Use `DEL` button or `Backspace` key to delete the last digit

## Files

- `index.html` - HTML structure
- `style.css` - Styling and layout
- `script.js` - Calculator logic and functionality
